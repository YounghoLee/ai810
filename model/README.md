# Data
## Data Format
* Pickle Data
* Dict
  * Key : 1 ~ N
  * Value : Tuple ( X ,Y )
    * X : N x M Tensor  ( M은 84로 고정, Mth Column : 시간 값)
    * Y : shock, mortality

## Train / Test
* trainData : 2857 / 1210 ( shock ) / 1946 ( mortality )
* testData  : 711 / 239 ( shock ) / 487 ( mortality )
